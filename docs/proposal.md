# 《鸡鸡大师》专业养成类游戏设计方案

作为一名专业的养成类游戏设计师，我针对目前《鸡鸡大师》的现状，从**情感连接、数值循环、仪式感、以及长线目标**四个维度给出优化建议。

---

## 1. 强化情感连接 (Emotional Bond)

目前角色只是一个“点击响应器”，缺乏生命属性。建议增加以下维度：

- **心情/饱食度系统**：
  - 增加一个简单的状态条。点击角色消耗体力，不点击一定时间会变“饿”或“不开心”。
  - 角色在不同状态下会有不同的对话气泡（例如：“今天运气真不错！”或者“我想吃灵石了...”）。
- **好感度等级 (Affection Level)**：
  - 通过互动（抚摸、使用技能）提升好感度。
  - 高好感度可以解锁更亲密的动画或专属的表情包。

---

## 2. 商业模式与病毒裂变循环 (Viral Loop & Economy)

为了实现快速获客并降低进入门槛，我们将初期重点放在**“病毒式裂变”**上，通过社交分享替代直接付费。

### 核心机制：以分享换资源
- **灵气获取 (Social Aura)**：当灵气耗尽时，玩家可以选择“求助好友”或“分享今日运势”，分享成功后立即补满灵气（每日限 X 次）。
- **运势勋章 (Sharing Posters)**：
  - 测算结果通过 AI 生成一张**极具美感、适合社交媒体传播**的“手相解析海报”或“今日签文”。
  - 海报上附带个人专属的小鸣叫二维码，扫码进入即可为分享者增加 1 点“灵气”。

### 社交驱动循环 (Viral Funnel)

| 阶段 | 动作 | 激励 | 目的 |
| :--- | :--- | :--- | :--- |
| **触达** | 玩家测算出“上上签”或罕见手相。 | 获得“炫耀勋章”特效。 | 触发分享欲望 |
| **爆发** | 玩家一键分享结果到朋友圈/群聊。 | 随机获得 10~50 鸡币或 5 灵石。 | 驱动主动传播 |
| **转化** | 好友通过分享图进入游戏。 | 分享者获得“永久灵气上限 +1”。 | 实现用户增量 |
| **留存** | 好友成为“同门成员”。 | 互点“喂食”，增加双方互动值。 | 建立社交关系链 |

### 极致简化的经济系统 (Simplified Economy)

为了让任何用户都能“秒懂”并快速开始游戏，我们抛弃复杂的双代币体系，采用**“单货币 + 灵气”**的最简模型：

### 1. 核心资源：鸡币 (Only One Currency)
- **唯一性**：不再区分付费代币和免费代币。
- **获取**：登录、分享、旅行带回。
- **用途**：解锁新职业、升级鸡舍装饰、购买高等级符咒。

### 2. 行动力：灵气 (Aura / Energy)
- **极简消耗**：使用任何功能点（如测算、出发旅行）消耗 1 点灵气。
- **极简恢复**：
  - **自动补给**：每 10 分钟自动回 1 点。
  - **呼朋唤友 (LIFT)**：分享一次到群聊，灵气瞬间补满（新手阶段最强驱动）。
  - **看广告补能量**：点击一下，看完广告，继续测算。

### 3. “小白级”转化漏斗 (UX Flow)

| 阶段 | 玩家感知 | 动作 | 目的 |
| :--- | :--- | :--- | :--- |
| **0 门槛上手** | “我想算个命” | 点击进入，免费赠送 5 次灵气。 | 建立第一印象 |
| **首次裂变** | “灵气用完了，还想算” | 点击“求签”，一键分享给好友即补满。 | 完成病毒传播 |
| **深度体验** | “想要更酷的外观” | 提示用积蓄的“鸡币”兑换，或小额直购“鸡币包”。 | 引入付费尝试 |

### 4. 商业化闭环 (Simplified Monetization)
- **广告为主 (Ads First)**：前期主要依靠“灵气恢复”和“高级奖励双倍”的视频广告。
- **订阅为辅 (Sub Later)**：提供“灵机月卡”，特权极其直观——**无限灵气 + 每日领鸡币**，彻底解决体力焦虑。

## 3. “灵鸡环游记”：放置类收集玩法 (The Traveling Chicken)

为了增加游戏的闲暇体验与长效收集目标，我们引入类似“旅行青蛙”的放置玩法：

### 核心玩法：说走就走的旅行
1. **准备行囊**：玩家在“鸡舍”中为主角准备食物（消耗鸡币）和幸运符。食物质量决定旅行时长，幸运符决定带回奖品的稀有度。
2. **主角离家**：主角会从主界面消失，进入“旅行中”状态。此时主界面会出现一张“我出门啦，记得想我~”的小纸条。
3. **明信片/特产回传**：旅行期间，玩家有概率收到主角从远方寄回的实时明信片（AI 生成的风景照，鸡鸡在其中摆拍）。

### 核心奖励：“鸡化”世界名画 (Chicken Masterpieces)
当主角旅行归来，会随机带回一张“世界名画”。这些名画经过了**“鸡化”处理**：
- **名画示例**：
  - 《蒙娜丽莎的微笑》 -> 《蒙娜鸡莎的神秘微笑》
  - 《戴珍珠耳环的少女》 -> 《戴珍珠项链的雏鸡》
  - 《梵高自画像》 -> 《戴着黑帽的算命鸡自画像》
- **文案增强 (New Copy)**：每张名画附带一段幽默的旅途感悟。例如：“在卢浮宫排队时，我发现艺术的真谛就是——不要让别人看清你的脚掌。”
- **收集图鉴**：玩家可以在“成就-名画馆”中查看所有收集到的艺术品。

### 裂变点 (Social Point)
- **艺术大展**：玩家可以将收集到的“名画”通过上述的**裂变海报**形式分享，展示自己的“高级审美”，吸引好友来互助增加下一次旅行的幸运值。

## 4. UI 视觉与布局重构建议 (UI/UX Optimization)

作为养成类游戏设计师，UI 不仅仅是按钮的堆砌，而是**场景的延伸**。我们将围绕“主角中心制”进行深度视觉重塑：

### 1. “重心锚定”布局 (Protagonist-Centric Layout)
- **视觉重心**：保持主角鸡在屏幕几何中心（45%-50% 高度），所有的 UI 元素应呈“环绕式”或“层级化”分布，避免遮挡角色。
- **信息分层**：
  - **顶层 (Status)**：极致轻量化的资源条，使用半透明“冰晶玻璃”质感。
  - **底层 (Context)**：角色下方的技能卡片应采用“浮岛”设计，增加呼吸灯特效，引导玩家点击。

### 2. “场景化”背景设计 (Scene-based Background)
- **从“渐变”到“空间”**：不再使用单一的线性渐变，而是构建一个**有深度的 2.5D 空间**：
  - **远景**：动态生成的流云、远山或随时间变换的星空/夕阳。
  - **中景**：角色站立的固定“展示台”或“草垫”，增加地面投影增强体积感。
  - **近景**：屏幕前偶有花瓣、羽毛或萤火虫飘过（使用 Canvas 粒子技术），增加灵动感。

### 3. 色彩体系：治愈系马卡龙 (Healing Palette)
- **主色调**：以“灵气蓝”和“鹅毛黄”为主，辅助色使用低饱和度的粉紫和嫩绿。
- **动态光影**：背景亮度随手机系统时间自动调节。中午色彩明亮饱和，深夜开启“静谧蓝”滤镜，契合养成类的治愈基调。

### 4. 交互微动作 (Micro-interactions)
- **视线追踪**：当玩家手指在屏幕上移动或悬停在某个按钮上时，主角鸡的头部会有微小的偏转，产生“它在看着我”的错觉。
- **软弹反馈**：按钮点击不再是生硬的缩放，而是带有“果冻感”的弹性形变。

## 5. “一职多能”系统实施方案 (Technical Path)

为了弱化职业的单一属性并增加游戏的策略深度，我们将重构系统，支持一个职业拥有多个技能。

### 类型定义变更
- `Career` 接口：将 `skill: CareerSkill` 替换为 `skills: CareerSkill[]`，并增加 `defaultSkillId: string`。

### 数据层重构
- 更新所有职业配置，将原有技能放入 `skills` 数组。
- 为“算命鸡”增加第二个示例技能（例如：八字精批）。
